//File Overview
solitaire.h: Header file containing the Solitaire class and method declarations.
solitaire.cpp: Implementation of the Solitaire class.
test.cpp: Unit tests to verify game functionality (deep copy, assignment operator, move validation).
How the Game Works
Board Initialization: Start the game by choosing a board configuration (Diamond or English).
Move Validation: Make moves based on the rules of Solitaire. Moves must jump over another marble into an empty hole.
Clearing and Restarting: You can clear the board and start a new game with the same or a different board shape.//
